Javascript sekos valdymas

Javascript reiškmės gali būti kelių skirtingų rūšių su skirtingomis savybėmis. Jos yra:

  • Skaičius - sveikasis arba su kableliu: 10, -15, 0.5
  • Tekstas - simboliai tarp kabučių, galima naudoti kabutes ', " arba `(šios turės papildomą savybę)
  • Boolean - true arba false
  • Undefined ir Null - reikšmės, nurodančios kad reikšmė nekeista arba kad duomenų nėra
  • Objektai ir masyvai - sudėtingesnės struktūros

Rašant funkcijas, dažnai reikia skirtingomis salygomis atlikti skirtingus veiksmus. Kad tą padaryti, reikia naudoti sekos valdymo struktūras - jos atlieka, praleidžia, kartoja veiksmus, jei atitinka nurodytos salygos.

Visoms joms reikalingos boolean reikšmės. Boolean yra reikšmė, kurios vienintelės galimos reikšmės yra true ir false.

Dėl paprastumo, kai tikimasi boolean tipo reikšmės, bandoma kitokias reišmes konvertuoti:

false

0 , -0, 0n

'', "", ``

null, undefined, NaN

true

Kiti skaičiai: 1, -100, 0.0001

Ne tuščia teksto eilutė: 'Hello', ' '

Bet koks objektas, masyvas ir t.t.

Dažniausiai naudojame boolean reikšmes ne tiesiogiai, bet gaunant ją iš palyginimo operacijų:

< > <= >=

Palyginimas tarp skaičių

Sekos valdymo struktūros yra trys: if-else, while-loop ir for-loop
