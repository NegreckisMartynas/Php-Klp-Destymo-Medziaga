Javascript sekos valdymas

Javascript reiškmės gali būti kelių skirtingų rūšių su skirtingomis savybėmis. Jos yra:

  • Skaičius - sveikasis arba su kableliu: 10, -15, 0.5
  • Tekstas - simboliai tarp kabučių, galima naudoti kabutes ', " arba `(šios turės papildomą savybę)
  • Boolean - true arba false
  • Undefined ir Null - reikšmės, nurodančios kad reikšmė nekeista arba kad duomenų nėra
  • Objektai ir masyvai - sudėtingesnės struktūros

Rašant funkcijas, dažnai reikia skirtingomis salygomis atlikti skirtingus veiksmus. Kad tą padaryti, reikia naudoti sekos valdymo struktūras - jos atlieka, praleidžia, kartoja veiksmus, jei atitinka nurodytos salygos.

Visoms joms reikalingos boolean reikšmės. Boolean yra reikšmė, kurios vienintelės galimos reikšmės yra true ir false.

Dėl paprastumo, kai tikimasi boolean tipo reikšmės, bandoma kitokias reišmes konvertuoti:

false

0 , -0, 0n

'', "", ``

null, undefined, NaN

true

Kiti skaičiai: 1, -100, 0.0001

Ne tuščia teksto eilutė: 'Hello', ' '

Bet koks objektas, masyvas ir t.t.

Su boolean reikšmėmis mes dirbame naudojant logines operacijas, taip kaip su skaičiais dirbame su artimetinėmis operacijomis. Jos yra:

&& - loginis AND

Rezultatas lygus true, jei abi pusės yra true:

true && true --> true

true && false --> false

false && true --> false

false && false --> false

|| - loginis OR

Rezultatas lygus true, jei viena ar abi pusės yra true:

true && true --> true

true && false --> true

false && true --> true

false && false --> true

! - loginis NOT

Dirba tik su viena reikšme, ją pakeičia priešinga:

!true--> false

!false--> true

Šių operacijų prioritetai: !, &&, ||

Kokie bus rezultatai?

true && false && true

true || false && true

(true || false) && true

!false && true

!(false && true)

Dažniausiai naudojame boolean reikšmes ne tiesiogiai, bet gaunant ją iš palyginimo operacijų:

> < >= <=

Daugiau/mažiau palyginimai tarp skaičių. Pridėtas lygybės ženklas reiškia, kad tinka ir vienodi skaičiai.

10 > 5 --> true

10 > 10 --> false

10 >= 10 --> true

Galima lyginti ir tekstą, bet rezultatas gali būti klaidinantis.

'abc' < 'cbd' --> true

'zebras' > 'citrina' --> true

'Zebras' > 'citrina' --> false

'zebras' > 'česnakas' --> false

=== !== == !=

Atitikimo palyginimai.

=== grąžina true, kai reikšmės abejose pusėse vienodos.

!== grąžina true, kai reikšmės abejose pusėse nevienodos

== ir != atlieka atitinkamus veiksmus, bet pirma pabando suvienodinti tipą. Recomendacija naudoti pirmas dvi operacijas veitoj šitų.

10 === 5 --> false

10 === 10 --> true

10 !== 5 --> true

10 === '10' --> false

10 == '10' --> true

10 == 'labas' --> false

Sekos valdymo struktūros yra trys: if-else, while-loop ir for-loop

Dažiausiai iš jų naudojama yra if-else, skirtas veiksmui atlikti tik esant nurodytai salygai. Sintaksė: 

                        
                            if(condition) {
                                //do if true
                            }

condition yra boolean reikšmė - jei ji true, vykdomas viduje esantis kodas, jei ne - praleidžiama.

Jei reikia atlikti kitą veiksmą, esant false reikšmei, pridedame ir else salygą: 

                        
                            if(condition) {
                                //do if true
                            }
                            else{
                                //do if false
                            }

Galiausiai, galima sujungti kelias if salygas, kad jas tikrinti iš eilės. Bus įvykdyta pirma tinkanti salyga arba else, jei nė viena netinka:

                        
                            function compare(number) {
                                if(number < 5) { 
                                    return 'Less than 5';
                                }
                                else if (number < 7) {
                                    return  'Less than 7';
                                }
                                else if (number < 11) {
                                    return  'Less than 11'; 
                                } 
                                else {  
                                    return  '11 or more';
                                }
                            }

Jei reikia veiksmą kartoti, reikia naudoti kartojimo struktūrą. Viena iš jų yra for-loop. Ją naudajame, kai žinome kiek kartų reikia veiksmą pakartoti. Jos sintaksė:

                        
                            for (pradinis_veiskmas; tikrinimo_salyga; veiksmas_po_kartojimo) {
                                //kartojamas veiksmas
                            }

Labai dažnai pradinis veiksmas ir veiksmas po kartojimo yra standartiniai ir for-loop atrodo taip:

                        
                            for (let i = 0; i < pakartojimu_skaicius ; i++) {
                                //kartojamas veiksmas
                            }

Kita kartojimo struktūra yra while-loop. Ją kartojame kai nežinome, kiek kartų reikės kartoti veiksmus. Jos sintaksė:

                        
                            while (salyga) {
                                //kartojame, kol salyga === true
                            }

Naudojant while-loop svarbu, kad vienas iš veiksmų keistų salygos reikšmes, nes jei salyga nekis, tai ciklas bus kartojamas be pabaigos. Taip pat galima ciklo viduje panaudoti break;, kuris užbaigia ciklą, arba jei esame funkcijoje, grąžinti reikšmę su return
