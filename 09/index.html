DOM, HTML elementų pasiekimas per Javascript

className

className yra HTML elemento atributas, kurio reikšmė: klasės teksto eilutėje. Jį pamodifikavus bus pakeistos ir elemento CSS klasės.

Tiesioginaime DOM manipuliavime vietoj jo rekomenduojama naudoti classList, bet jis reikalingas nustatyti klasėms naudojant React. 

const classes = document.querySelector('.example-class').className; // 'example-class foo

classList

classList yra "live" elemento klasių kolekcija. Jos naujos reikšmės nustatyti negalime, bet jos reiškmes patogu redaguoti naudojant jos metodus. Šie metodai yra:

add(className), remove(className), contains(className), toggle(className), replace(oldClass, newClass)

document.createElement(tag)

Sukuria naują elementą su nurodytu tag ir jį grąžina. Šis elementas yra nėra rodomas, kol jis nėra įdėtas į egzistuojantį elementą.

createElement metodą turi tik document elementas - jei bandome kviesti medotą nuo kitų elementų gausime klaidą

element.innerHTML

Grąžina elemento turinį kaip tekstą, pakonvertavus vidinius elementus į HTML tekstą.

Pakeičiant šito elemento reikšmę duotas tekstas but interpretuotas kaip HTML elementai. Tai vienas iš būdų sukurti naujus HTML elementus.

element.append(newElement)

Įdeda newElement į element vidinius elementus pabaigoje. Taip pat galima vietoj newElement duoti string: tada į element pabaigą bus pridėtas duotas tekstas.

element.prepend(newElement)

Įdeda newElement į element vidinius elementus pradžioje. Taip pat galima vietoj newElement duoti string: tada į element pradžią bus pridėtas duotas tekstas.

element.remove()

Pašalina elementą iš tėvinio elemento, bet jo nesunaikina, kol egzistuoja kintamasis, jį saugojantis. Po to pašalintą elementą galima įdėti į kitą elementą.

element.parentElement

Grąžina tėvinį elementą. Su metodais querySelector galime eiti gilyn, o su parentElement- grįžti atgal.
