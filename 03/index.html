Javascript masyvai ir objektai

Kartais norime viename kintamajame saugoti kelias reikšmes; tai galime padaryti saugojant šias reikšmes masyvuose ar objektuose.

Masyvai ir Objektai yra sąrašai su indeksu, kur kiekvienas indeksas nurodo atitinkamą reikšmę.

Masyvai

                        const fruits = ['Apple', 'Banana', 'Pear'];
                            fruits[0]; --> 'Apple' 
                            fruits[1]; --> 'Banana' 
                            fruits[2]; --> 'Pear' 
                            fruits[3]; --> undefined
                            fruits[2] = 'Plum';
                            fruits[2]; --> 'Pear'
                            fruits.length; --> 3

Masyvai - reikšmių sąrašai su eilės tvarka. Jie tiesiogiai kuriami tarp [ ] dedant reikšmes, skiriant jas kableliu.

Masyvo indeksai - numeriai, pradedant nuo 0 didėjimo tvarka, t.y. 0, 1, 2, 3 ir t.t. Jo reikšmes galima pasiekti prie kintamojo pavadinimo pridedant [ ] ir į jų vidų parašius jų indeksą.
Pvz: fruits[2] paims trečią elementą, nes indeksacija prasideda nuo 0.

Masyvai turi savybę length, kuri pasako kiek elementų jame yra. Jos reikšmė bus vienu daugiau nei didžiausias indeksas.

Masyvai ir for-loop

                        const fruits = ['Apple', 'Banana', 'Pear']
                              for (let i = 0; i < fruits.length ; i++) {
                                  const fruit = fruits[i]
                                  console.log('Next fruit is ' + fruit)
                              }

Dažniausias for-loop panaudojimas: eiti per visas masyvo reikšmes ir su jomis vykdyti komandas. Tai įprastai daroma naudojant pradinę sąlygą 0, palyginimo operaciją su masyvo length, veiksmas po kartojimo didinantis iteratorių vienetu. (let i = 0; i < array.length ; i++)

Mokant originalų metodą bus galima jį pritaikyti kitokioms situacijoms, toliau išmoksime kitus metodus iteruoti masyvą.

forEach ir for...of

Vietoj for-loop, masyvams iteruoti galime panaudoti funkciją forEach arba struktūrą for...of

                        const fruits = ['Apple', 'Banana', 'Pear'];
                              for (let i = 0; i < fruits.length ; i++) {
                                  const fruit = fruits[i];
                                  console.log('Next fruit is ' + fruit);
                              }
for...of:
                        const fruits = ['Apple', 'Banana', 'Pear'];
                              for (let fruit of fruits) {
                                  console.log('Next fruit is ' + fruit);
                              }
forEach:
                        const fruits = ['Apple', 'Banana', 'Pear'];
                              fruits.forEach( (fruits) => {
                                  console.log('Next fruit is ' + fruit);
                              })
arba 
                        const fruits = ['Apple', 'Banana', 'Pear'];
                              const printFruit = (fruit) => console.log('Next fruit is ' + fruit);
                              fruits.forEach(printFruit);

Objektai

                        const student = {firstName: 'Vardenis', lastName: 'Pavardenis', grades: 7.5};
                            student.firstName; --> 'Vardenis' 
                            student.lastName; --> 'Pavardenis' 
                            student.grades; --> 7.5 
                            student.other; --> undefined
                            student.grades = 8.2;
                            student.grades; --> 8.2

Objektai - reikšmių žodynas be eilės tvarkos. Jie tiesiogiai kuriami tarp { } rašant įrašus - reikšmės raktus ir jų reikšmes - perskyrus dvitaškiu, įrašus perskiriant kableliu

Objekto indeksai - žodiai: dažniausiai be visų neraidinių ar skaitinių simbolių. Jo reikšmės dažniausiai pasiekiamos po kintamojo pavadinimo dedant tašką ir tada raktą.

Raktai gali būti bet koks tekstas, bet jei yra tarpų arba kitokių simbolių, kuriant objektą raktą reikia įrašyti į kabutes (const obj={'example-prop': 'Labas'}). Tada įrašas pasiekiamas panašiai, kaip ir masyve: obj['example-prop']

Dažnai susidursite su JSON tipo duomenimis. Šio tipo duomenys atitinka JS objectus, su pagrindiniu skirtumu, kad raktai privalo būti su kabutėmis. JSON į JS objektą galima konvertuoti naudojant JSON.parse(jsonString) metodą, o objektą į JSON galima konvertutoti naudojant JSON.stringify(object) metodą.

Masyvas vs. objektas

Masyvus naudoti, kai
  • Svarbi eilės tvarka
  • Atliekame tokius pačius veiksmus su visais elementais
  • Individualūs įrašai neturi išskirtinės prasmės
Objektus naudoti, kai
  • Eilės tvarka nesvarbi
  • Įrašų reikšmės yra skirtingų tipų
  • Kiekvieno įrašo reikšmė turi aiškią prasmę
  • Magiški triukai (ciklai)
  • Paukščių stebėjimas
  • Rezultatų lenta
