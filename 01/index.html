Javascript pagrindai

Javascript yra interpretuojama programavimo kalba, skirta interneto puslapių funkcionalumo praplėtimui. Be Javascript vieninelės interakcijos su puslapiu yra navigacija bei formų siuntimas. Javascript veikia viduje naršyklės toks, koks jis atkeliauja iš serverio - su serveriu susisiekti jau reikalingi specialūs metodai.

Toliau mes naudosime integruotą naršyklės konsolę, kurioje ir rašysime Javascript. Čia parašytas Javascript nebus išsaugotas, tad jei pridarome klaidų, kurių nebepataisome, galime tiesiog perkrauti langą ir rašyti iš naujo.

Kad atidaryti konsolę, dešiniu spaudžiame ant puslapio fono ir renkamės inspect element, tada atsidariusioje sekcijoje pasirenkame Console tab.

Čia pabandykite parašyti 2 + 2 ir paspausti Enter : gausite rezultatą 4

Šitoje konsolėje galite atlikti visus veiksmus, galimus padaryti su Javascript kodu: atlikti skaičiavimus, kviesti funkcijas ir metodus, modifikuoti puslapyje esančius objektus ir HTML elementus

Pabandykite dar kelias operacijas:

6 * 7

155 / 10

4 ** 2

'Hello' + 'World'

Po eilutės įvykdymo rezultatas dingta, nes jis nėra priskirtas kaip kintamojo reikšmė. Kad iš išsaugoti reikšmę, reikia sukurti kintamajį su pavadinimu:

> const result = 2 + 2;
> result;
< 4

Kintamasis - vardas, kuriam galima priskirti reikšmę ir kuriuo lengvai galime tą reikšmę pasiekti.

const reiškia, kad result reikšmė visada išliks tokia pat. Vietoj const galite panaudoti let: jis reiškia, kad kintamojo reikšmė gali būti pakeista.

Rekomendacija pirmenybę teikti const, o let naudoti išimtinai, kai keitimas yra reikalingas.

Be const ir let dar yra ir var: jis atlieka tokią pačią funkciją kaip ir let, bet jis buvo sukurtas pirmiau ir turi neintuityvią interakciją su scopes; let sukurtas kaip var pataisymas ir var paliktas toks pat dėl seno kodo palaikymo.

Javascript vykdomas nuo viršaus į apačią iš eilės. Dažnai reikalinga, kad kažkokia kodo dalis būtų vykdoma ne iš karto, o mūsų norimu metu. Tai mes galime padaryti sukuriant funkciją.

                        
                            function helloWorld() {
                                return "Hello World!";
                            }

Funkcijos apibrėžia, kokius veiksmus atlikti ir kokį rezultatą grąžinti, kai ji iškviesta. Funkcijos pavadinimas tampa kintamuoju, o kad ją iškviesti prie pavadinimo reikia pridėti skliaustelius.

> helloWorld();
< "Hello World!"

Šitą funkciją galite rašyti konsolėje (Eiti į kitą eilutę: Shift + Enter) arba galite ją parašyti šio HTML failo <script> ir perkrovus puslapį ją iškviesti konsolėje.

Kita galima funkcijos sintaksė:

                            
                                const helloWorld = () =>  "Hello World!";
                            
                             arba 
                            
                                const helloWorld = () =>  { return "Hello World!";};

Funkcijos gali turėti parametrų, kurių reikšmes užpildome funkciją kviečiant:

                        
                            function suma(first, second) {
                                const suma = first + second;
                                return suma;
                            }

> suma(10, 15);
< 25

Prie funkcijos užrašyti pavadinimai ( first, second ) vadinami parametrais, o jų reikšmės vykdymo metu (10, 15) - argumentais. 

                                
                                    const suma = (first, second) => {
                                        const suma = first + second;
                                        return suma;
                                    }
