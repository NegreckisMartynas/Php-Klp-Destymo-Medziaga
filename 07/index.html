Objektų konstrukcija, klasės

Konstavimas

Panašios struktūros objektus kurti galime parašyti tos paskirties funkcijas.

Grąžinamas rezultatas: objektas su savybėmis ir metodais.


                        function student(firstName, lastName) {
                            return {
                                firstName: firstName,
                                lastName: lastName,
                                grades: [],
                                addGrade(grade) {
                                    this.grades.push(grade);
                                },
                                toString() {
                                    return `Student ${this.firstName} ${this.lastName}`;
                                }
                            }
                        }

Class

Klases aprašymas apibūdina, kaip sukurti naujus objektus.

constructor metodas nustato objekto pradinę būseną pagal duotus parametrus.

Metodai ir savybės rašimi tame pačiame lygyje, be jokių keywords.

Klasės pavyzdys:


                        class Student {
                            grades = [];
                            constructor(firstName, lastName) {
                                this.firstName = firstName;
                                this.lastName = lastName;
                            }
                        
                            addGrade(grade) {
                                grades.push(grade)
                            }
                        
                            toString() {
                                return `Student ${this.firstName} ${this.lastName}`;
                            }   
                        }
                        
                        const newStudent = new Student('Vardenis', 'Pavardenis');
