Masyvų metodai ir atributai

foreach((element, index, array) => {})

Per masyvą iteruojanti funckija, kuri nieko negrąžina.

Ją naudoti, kai po veiksmo nebereikia rezultato: duomenų išvedimas, išorinės funkcijos kvietimas ir t.t.

Šio ciklo užbaigti ankščiau su return negalima. Jei to reikia, naudojame paprastą for-loop arba for-of

element yra dabartinis masyvo elementas, pradedant nuo [0].

index yra elemento indeksas masyve, pradedant nuo 0.

array yra pradinis masyvas. Rekomenduojama jo nemodifikuoti funkcijos viduje. 


                                [5, 1, 3, 4, 5, 7].forEach(
                                    element => console.log(element)
                                )
                                

                                const array = [5, 1, 3, 4, 5, 7];
                                for(let i = 0; i $lt; array.length; i++){
                                    const element = array[i];
                                    console.log(element);
                                }

map((element, index, array) => {return transformedElement})

Per masyvą iteruojanti funckija, kuri grąžina naują masyvą, pakonvertavus elementus pagal funkciją.

Kiekvienam elementui pritaikoma duota funkcija ir jos rezultatas yra idedamas į rezultatų masyvą ta pačia tvarka.

Rezultatas visada turės tiek pat elementų, kiek pradinis masyvas.

element yra dabartinis masyvo elementas, pradedant nuo [0].

index yra elemento indeksas masyve, pradedant nuo 0.

array yra pradinis masyvas. Rekomenduojama jo nemodifikuoti funkcijos viduje. 


                                const result = [10, 20, 30, 40, 50].map(
                                    element => element * 10
                                );
                                result === [100, 200, 300, 400, 500];
                                

                                const array = [10, 20, 30, 40, 50];
                                const result = [];
                                for(let i = 0; i $lt; array.length; i++){
                                    const element = array[i];
                                    result.push(element * 10);
                                }
                                result === [100, 200, 300, 400, 500];

filter((element, index, array) => {return true || false})

Per masyvą iteruojanti funckija, kuri grąžina naują masyvą, paėmus tik elementus, kurie grąžino true.

Rezultatas gali turėti tiek pat elementų, mažiau arba 0, bet rezultatas visada bus masyvas.

element yra dabartinis masyvo elementas, pradedant nuo [0].

index yra elemento indeksas masyve, pradedant nuo 0.

array yra pradinis masyvas. Rekomenduojama jo nemodifikuoti funkcijos viduje. 


                                const result = [10, 25, 33, 40, 54].map(
                                    element => element % 10 === 0
                                );
                                result === [10, 40];
                                

                                const array = [10, 20, 30, 40, 50];
                                const result = [];
                                for(let i = 0; i $lt; array.length; i++){
                                    const element = array[i];
                                    if(element % 10 === 0) {
                                         result.push(element);
                                    }
                                }
                                result === [10, 400];

reduce((prevElement, nextElement, index, array) => {return prevElement}, initialValue)

Per masyvą iteruojanti funckija, kuri sutraukia rezultatus į vieną.

Su šia funkcija galima padaryti labai daug skirtingų veiksmų. Paprasčiausias ir lengviausias yra sumavimas:


                        const result = [1, 2, 3, 4].reduce(
                            (a, b) => a+b
                        );
                        result === 10;

Po kiekvieno ciklo, grąžinta reikšmė padudama kaip prevElement kitam ciklui. Pabaigus prevElement grąžinamas kaip rezultatas.

prevElement yra pirmasis elementas arba kaupiamoji reikšmė.

nextElement yra elementas po prevElemento.

index yra elemento indeksas masyve, pradedant nuo 0.

array yra pradinis masyvas. Rekomenduojama jo nemodifikuoti funkcijos viduje.

initialValue yra pradinė prevElement reikšmė: jei ji yra, tai nextElement = array[0]; jei ne, tai prevElement = array[0] ir nextElement = array[1]


                                const sum = [10, 25, 33, 40, 54].reduce(
                                    (sum, element) => sum + element, 0
                                );
                                sum === 162;
                                

                                const array = [10, 20, 30, 40, 50];
                                const sum = 0;
                                for(let i = 0; i $lt; array.length; i++){
                                    const element = array[i];
                                    sum = sum + element;
                                }
                                sum === 162;

Galima įsivaizduoti, kad reduce sutraukia pirmas dvi masyvo reikšmes į vieną iki kol lieka tik viena:

[10, 25, 33, 40, 54].reduce((sum, element) => sum + element, 0)

[0, 10, 25, 33, 40, 54]

[10, 25, 33, 40, 54]

[35, 33, 40, 54]

[68, 40, 54]

[108, 54]

[162]

Naudojant reduce, galima padaryti sudėtingesnes operacijas, pavyzdžiui, išskaidyti masyvo skaičius į lyginius ir nelyginius:


                        const oddEven = [10, 25, 33, 40, 54].reduce(
                            (oddEven, element) => {
                                 if(element % 2 === 0){
                                     oddEven[1].push(element)
                                 } else {
                                     oddEven[0].push(element)
                                 }
                                 return oddEven;
                            }, [ [],[] ]
                        );
                        const [odd, even] = oddEven;
                        odd === [25, 33];
                        even === [10, 40, 54];
